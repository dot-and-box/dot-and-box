# Dots

Dots Control

Dots makes declaring multiple dots easier

```dabl tab showLineNumbers
title: 'dots'
dots ids: 1 2 3 4 5 6 at: [-3,-0.5]
```

## dots attributes

- group - prefix for ids, optional
- ids - space separated ids of dots, mandatory
- layout - _col_ or _row_, optional - default: col
- span - number of empty cells between dots
- colors - space separated color list used to fill dots
- radius - dot radius

:::note
if _id_ is set, effective control id is concatenation of _id_ "\_"
and identifier from _ids_
e.g. declaration:
dots id: pre ids: 1 2 3
makes three dots with id:
pre_1 pre_2 pre_3
:::
